# Java Calculator (OOP)

A menu-driven Java calculator using Object-Oriented Programming concepts.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Menu-driven program
- OOP design (Classes & Objects)

## How to Run

```bash
javac Calculator.java Main.java
java Main
```

## Advanced Features

- Interface-based design
- Exception handling using try-catch
- Polymorphism
- Clean OOP architecture
